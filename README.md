# 📝 Koushik's AI Summarizer

A lightning-fast, cloud-hosted AI tool built with Python and Streamlit. 

This application uses the Groq API and Llama 3 to instantly summarize long articles, emails, or reports into clean, actionable bullet points. 

### 🚀 Tech Stack
* **Frontend:** Streamlit
* **Backend:** Python
* **AI Inference:** Groq (Llama 3)
